Stride Shader Basics

Getting Started

Welcome to Stride shader programming! Here are the fundamental concepts:

Shaders in Stride use HLSL (High-Level Shading Language) with some Stride-specific extensions. Detailed

Basic Shader Structure

shader MyShader : ComputeColor
{
    override float4 Compute()
    {
        return float4(1, 0, 0, 1); // Red color
    }
};

Common Examples

Solid Color

shader SolidBlue : ComputeColor
{
    override float4 Compute()
    {
        return float4(0, 0, 1, 1); // Blue
    }
};

PulsingColor

shader PulsingColor : ComputeColor
{
    override float4 Compute()
    {
        float pulse = sin(time * 2) * 0.5 + 0.5;
        return float4(pulse, 0, 1-pulse, 1);
    }
};
